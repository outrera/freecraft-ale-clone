What is ALE Clone?

A clone of the realtime fantasy strategy game W*rCr*ft II.
ALE Clone runs under UNIX and X or Win32.

Since ALE Clone doesn't provide graphics, sounds and documentation,
you need the original game so ALE Clone can extract the necessary files. (With over 1 million sold copies this shouldn't be a big problem)
